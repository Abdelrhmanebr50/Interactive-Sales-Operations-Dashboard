# Interactive Multi-Page Sales & Operations Dashboard (ZARA Case Study)

## 📌 Project Overview
This project focuses on analyzing sales data for ZARA to extract actionable insights regarding product performance, seasonal trends, and revenue optimization.

## 🛠️ Tools Used
- **Microsoft Excel:** Data Cleaning, Pivot Tables, and Dashboard Design.
- **Data Visualization:** Interactive slicers and multi-page layout for better user experience.

## 📊 Key Insights
- **Top Categories:** Analysis of which products (Jackets, Shoes, etc.) drive the most revenue.
- **Seasonal Trends:** Identifying peak sales periods (Winter vs. Summer collections).
- **Promotion Impact:** Evaluation of how promotions affect total sales volume.

## 📁 Repository Structure
- `/Data`: Contains the dataset used for analysis.
- `/Dashboard`: The final Excel workbook with the interactive dashboard.
